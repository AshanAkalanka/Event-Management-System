# 🎉 Event Management System

A full-stack **Event Management System** developed as a **Second Year, First Semester Software Engineering module project** which is sub-module of the Catering Management System.  
This project is built using **Java Spring Boot**, **MySQL**, **HTML**, **CSS**, and **JavaScript**.

The system is designed to help users manage events in an organized and user-friendly way. It includes both **customer-side** and **admin-side** features, along with authentication, profile management, and live chat support.

My main contribution to this project is the **Event Management module**, which handles event-related operations and coordination features.


## 📚 Academic Project Information

- **Module**: Software Engineering
- **Year**: Second Year
- **Semester**: First Semester
- **Project Type**: Group Full-Stack Web Application
- **My Contribution**: Event Management Module


## ✨ Project Overview

The Event Management System allows users to register, log in, manage their profiles, create and manage event details, and communicate with the admin through live chat.  
The admin side also provides management features to monitor users, events, and system activities.

This system was developed to improve event handling, reduce manual work, and provide a better user experience through a centralized web platform.

## 🏗️ System Architecture

The project follows a full-stack web application structure:

- **Frontend** handles the user interface using HTML, CSS, and JavaScript
- **Backend** handles business logic, authentication, event management, and admin functions using Spring Boot
- **Database** stores user data, event details, chat data, and admin-related records in MySQL
