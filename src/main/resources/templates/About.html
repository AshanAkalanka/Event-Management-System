About

I created this project for my 2nd Year 1st Semester Software Engineering module. It is a web-based online catering services system, and my responsibility is the Event Management component.

This system is developed using Java Spring Boot for backend functionality and HTML, CSS, and JavaScript for the frontend interface.

Project Overview

Event Creation & Management

The system allows users to create, update, and manage catering events efficiently. All event details including date, location, event type, and customer requirements are handled in an organized way.

Status Tracking

Events can be tracked based on their status such as planned, confirmed, completed, or canceled, improving workflow and management visibility.

Technology Stack

Spring Boot handles backend logic and data operations, while the frontend provides a clean, responsive, and user-friendly design.

Goal & Experience

The goal is to build a professional-looking system that simplifies event planning and improves catering management efficiency.
